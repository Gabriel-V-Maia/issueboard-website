// auth

Autenticação em 5 passos

  1. 01
    Iniciar login

    Você clica em login no app desktop.

  2. 02
    Device code

    O app solicita um device_code ao GitHub.

  3. 03
    Autorização

    Você autoriza o acesso pelo navegador.

  4. 04
    Polling

    O app aguarda o token via polling na API do GitHub.

  5. 05
    Token salvo

    Credenciais persistidas em ~/.issueboard/config.json.
