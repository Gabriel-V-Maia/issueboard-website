Features

Tudo que você precisa

OAuth

Autenticação Device Flow

Login seguro com GitHub sem precisar criar tokens manualmente. Autorize pelo navegador em segundos.

Cache

Cache inteligente

Issues são cacheadas por 5 minutos para evitar rate limiting e tornar o uso mais ágil no dia a dia.

Kanban

Board três colunas

Open, In Progress e Done. Classificação automática por label e padrões no título da issue.

Multi-repo

Múltiplos repositórios

Agrega issues de todos os seus repos em um único lugar, deduplicadas e organizadas.

Busca

Padrões automáticos

Detecta TODO, WIP, FIXME no título e labels como wip, doing, todo automaticamente.

Async

Carregamento assíncrono

Interface responsiva com threads para buscar issues sem travar a UI enquanto você trabalha.

Como funciona

Autenticação em 5 passos

01

Você inicia o login no app

02

App solicita um device_code ao GitHub

03

Você autoriza via navegador

04

App faz polling até receber o token

05

Token salvo em ~/.issueboard/
